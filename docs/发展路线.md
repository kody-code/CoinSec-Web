# CoinSec-Web 发展路线

> 基于 2026-06 项目现状梳理

## 项目现状

- 版本：v1.0.3
- 技术栈：Vue 3.5 + TypeScript 6 + Vite 8 + Element Plus 2.14 + Pinia + Vue Router 4 + Chart.js 4 + Capacitor 8
- 后端 API：25 个端点（认证/账单/分类/账户/转账/用户/图标）
- 前端已对接全部后端 API，唯一缺口：头像上传（API 已调用，Profile 页面已实现）

### 已完成功能

| 模块 | 功能 |
|------|------|
| 登录/初始化 | 玻璃态设计、浮动动画、首次初始化、自动跳转 |
| 仪表盘 | 双布局（Web/原生）、统计卡片、账户概览、近期记录、分类排行、FAB |
| 账单列表 | 无限滚动、类型筛选、日期筛选、转账弹窗、分页 |
| 记账表单 | 创建/编辑双模式、类型切换、分类网格、默认账户、日期/时间选择器 |
| 统计 | 日期预设、环形图、分类条形图、分类明细进度条 |
| 账户管理 | 渐变卡片、图标选择器、CRUD、删除级联提示 |
| 分类管理 | 收支切换、CRUD |
| 个人中心 | 昵称修改、密码修改、头像上传、默认账户偏好 |
| 布局 | Web 可折叠侧边栏 + 移动抽屉 / 原生底部 Tab + 页面过渡动画 |
